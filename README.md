## Solder tips

 * RFM95 module
 * Capacitor for 3V3-GND optional
 * D3 jumper, leave open while programming ESP8266
 * SMA edge connector for antenna, or use a wire
 * Dual base shield optional
 
## Unused D1 mini pinouts

 * A0
 * D4
 * TX, RX
 * RST
 * 5V
